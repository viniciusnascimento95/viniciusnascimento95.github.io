## Getting Started

### Prerequisites

- A modern web browser
- [Node.js](https://nodejs.org/) (for local development)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/viniciusnascimento95/viniciusnascimento95.github.io.git
   cd viniciusnascimento95.github.io
   ```

2. Install dependencies (optional, for local development):
   ```bash
   npm install
   ```

### Running Locally

To run the website locally, you can use any static file server. Here's how to do it with Node.js:

1. Install `serve` globally (if you don't have it):
   ```bash
   npm install -g serve
   ```

2. Start the server:
   ```bash
   serve .
   ```

3. Open your browser and go to `http://localhost:3000`

### Deployment

The website is automatically deployed to GitHub Pages whenever changes are pushed to the `main` branch.

## Usage

### Switching Languages

- Click the **PT** button in the navbar to switch to Portuguese
- Click the **EN** button (in Portuguese version) to switch to English
- The website will remember your language preference

### Navigation

- Use the navbar to jump to different sections
- Click on project links to view external websites
- Use the contact links to send an email or view LinkedIn profile
